HOW TO PLAYA=BACK
GOAL
Reduce your opponent to 0 GDP 3 times. Each collapse = 1 fall. 3 falls = eliminated.
YOUR TURN
Roll 3 dice up to 4 times. Build a WAR queue to attack and a DEF queue to defend. Both players resolve damage after both turns.
ACTIVATION
⊕ (FIST) activates symbols.
1 fist + 1 ATK = 1x WAR output.
1 fist + 1 DEF = 1x DEF absorb.
2 fists + 1 symbol = 2x activation.
Alien @ is self-activating — no fist needed.
COMBO BONUSES
ATK ATK + = 1x WAR queued.
ATK ATK (no x) = +2 WAR bonus!
DEF DEF + = 1x DEF queued.
DEF DEF (no x) = +2 DEF bonus!
Triple same = +3 flat bonus token.
DAMAGE
Output = (WAR stat x attacks queued) + bonus.
Absorbed = (DEF stat x defences queued) + bonus.
GDP lost = Output - Absorbed.
0 EPIDEMIC
2 epidemics in one turn ends it immediately. Feudal Lord and Alien Race carry more epidemic risk.
